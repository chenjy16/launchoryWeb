Privacy Policy

Last Updated: January 2, 2026

Launchory is designed with privacy in mind. This privacy policy explains how we handle your data when you use the Launchory macOS application.

1. Data Collection and Usage

Launchory is designed as a privacy-first utility. We do not collect, store, or transmit your personal data, app usage statistics, or file system information to external servers.

  • Local Processing: All application scanning, icon rendering, and layout organization occur locally on your device.
  • App Data: Launchory reads the list of applications installed on your Mac solely to display them within the launcher interface. This list is never shared.
  • Preferences: User preferences (such as theme choices, icon sizes, and folder layouts) are stored locally on your Mac using standard macOS configuration files.

2. Permissions

To function correctly, Launchory requires specific permissions from macOS:

  • Application Data: To verify installed applications and launch them.
  • Accessibility (Optional): Used solely to support game controller input for navigation.

3. Third-Party Services

Launchory does not use any third-party analytics SDKs, tracking pixels, or advertising frameworks.

4. Children‚Äôs Privacy

Launchory does not knowingly collect data from children, as it collects no data at all.

5. Changes to This Policy
